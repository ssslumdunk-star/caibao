📊 美股财报日历

企业级的实时财报数据系统，集成智能更新调度器和价格验证机制

⚠️ 部署说明

这是一个GitHub Pages静态页面。要运行完整的财报日历应用，需要在本地环境或服务器上部署Python Flask应用。

🤖 全自动化更新

智能调度器每个交易日自动更新股价和财报数据，无需手动干预

📊 价格验证机制

自动检测异常股价，防止Oracle类似的数据错误，确保数据准确性

🎯 真实分析师评论

来自摩根士丹利、高盛等顶级投行的专业分析师观点，非模板生成

🚀 三层架构设计

Demo版(5001) / 缓存版(5002) / 实时版(5000)，满足不同场景需求

💰 多市场支持

美股 + 港股数据，自动货币转换，覆盖主要股票市场

📦 一键部署

完整的自动化设置脚本，包含监控、日志和错误恢复机制

🛠️ 本地部署指南

1. 克隆项目

git clone https://github.com/ssslumdunk-star/caibao.git
cd caibao

2. 安装依赖

pip3 install flask requests beautifulsoup4 sqlite3 schedule

3. 一键启动

# 设置自动化系统
./setup_auto_updates.sh

# 启动Web服务器
python3 cached_earnings_server.py

4. 访问系统

  • 主服务: http://localhost:5002
  • Demo版: http://localhost:5001
  • API状态: http://localhost:5002/api/cache_stats
